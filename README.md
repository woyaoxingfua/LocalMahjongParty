# 局域网麻将游戏

## 项目概述
一个基于Web技术的局域网麻将游戏系统，支持4人实时对战，采用前后端分离架构，通过WebSocket实现实时通信，适配桌面和移动设备。

## 核心功能
- 支持碰、杠、胡等完整麻将规则
- 4人自动房间匹配，集齐后自动开始游戏
- 本地存储玩家ID，支持断线重连
- 响应式设计，适配桌面端与移动端
- 使用Unicode麻将牌字符，模拟真实视觉效果

## 技术栈
- **后端**：Python、Flask、Flask-SocketIO
- **前端**：HTML5、CSS3、JavaScript、Socket.IO客户端
- **依赖**：eventlet（异步网络引擎）

## 安装与使用
### 前置条件
- Python 3.6+
- 所有玩家需处于同一局域网

### 安装步骤
1. 克隆或下载项目到本地
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 启动游戏
1. 启动服务器：
   ```bash
   python server.py
   ```
2. 玩家在浏览器访问：`http://<服务器IP地址>:5000`（本地可直接访问`http://localhost:5000`）
3. 集齐4名玩家后，游戏自动开始

## 游戏操作
- **摸牌**：轮到自己回合时自动摸牌
- **出牌**：选择手中的牌点击即可打出
- **操作选择**：当可进行碰、杠、胡操作时，点击对应按钮或"过"
- 界面显示当前回合玩家、手牌、其他玩家牌数及已打出的牌
