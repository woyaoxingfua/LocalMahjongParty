# 局域网麻将 — 项目概览

## 当前版本：v3（2026-05-24 第二轮深度优化）

### 项目结构

```
majiang_system/
├── server.py        40行  入口（Flask + SocketIO 初始化 + 启动）
├── events.py       180行  SocketIO 事件处理（胶水层）
├── room_manager.py 142行  房间/玩家全局注册表（单例）
├── game.py         380行  MahjongGame 状态机
├── logic.py        200行  纯算法（胡牌、向听数、进张、国士）
├── tiles.py         84行  牌面定义、排序、Unicode 映射
└── templates/
    └── index.html  ~800行  前端（CSS + JS，全自包含）
```

### 核心功能清单

#### 游戏规则
- [x] 四人麻将（东南西北，庄家轮转）
- [x] 胡牌判断（标准4副+对 + 七对子 + 国士无双向听）
- [x] 荣和 / 自摸 / 天胡
- [x] 碰 / 明杠 / 暗杠 / 补杠
- [x] 抢杠胡（补杠时其他玩家可荣和）
- [x] 岭上开花（杠后摸牌可和）
- [x] 荒牌处理（牌山耗尽流局）
- [x] 再来一局（庄家自动轮转）

#### 积分系统
- [x] 荣和：放炮者 -1000，胡牌者 +1000
- [x] 自摸：其余每人 -500，胡牌者 +1500
- [x] 结束弹窗展示本局变动 + 累计积分

#### 前端体验
- [x] 麻将桌四方位布局（上/左/右/自己）
- [x] 出牌二步确认 + 手机触摸上滑直接出牌
- [x] 进张列表提示（听牌时显示哪张牌能和）
- [x] 摸牌高亮 / 听牌动态发光
- [x] 碰/杠/胡浮动大字特效 + 音效（Web Audio 无外部文件）
- [x] 操作弹窗 15 秒倒计时条
- [x] 对手面板当前出牌轮金色高亮
- [x] 弃牌池按4家分栏显示
- [x] 消息日志（时间戳 + 自动折叠展开）
- [x] 连接状态 + 断线重连

### 启动方法

```bash
cd majiang_system
python server.py
# 局域网地址会自动打印，其他人用手机/电脑打开即可
```

### 依赖

```
flask
flask-socketio
eventlet
```

### 已知限制

- 仅实现基础积分（无番型、无役种计算）
- 国士无双支持向听显示，尚未实装和牌胡牌触发
- 7 对不重复对子七对子已支持，连对（4张同牌当两对）暂不支持
